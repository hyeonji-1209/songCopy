# songCopy 기능 총람

> Songsterr 클론 + 자체 개선 기능. 갱신일: 2026-07-08
> 라이브 데모: **https://songcopy.vercel.app** (읽기 전용 — 계정·저장 기능은 로컬/컨테이너 배포에서 동작)

---

## 1. 홈 (곡 카탈로그)

- **곡 목록** — 서버 DB에서 로드, 제목·아티스트·악기 태그 표시
- **검색** — 제목/아티스트 서버 패턴 검색 (입력 디바운스)
- **악기 필터** — 전체 / 기타 / 키보드 / 베이스 / 브라스 / 드럼
- **리비전 수 배지** — 커뮤니티 수정이 있는 곡 표시 (주황 태그)
- **파일 열기** — Guitar Pro / MusicXML 파일을 플레이어로 바로 열기

## 2. 플레이어 (곡 페이지)

### 악보
- **오선보 + 탭 동시 렌더링** (alphaTab, SVG) — 표기 모드 전환 가능: 탭+오선보 / 탭만 / 오선보만
- **섹션 라벨** (벌스, 코러스…), 템포·튜닝·박자표 표시
- **재생 커서** — 연주 중인 음표 실시간 초록 하이라이트 + 자동 스크롤
- **드럼 전용 퍼커션 표기** (하이햇 ×, 킥/스네어), 키보드는 오선보
- 악보 **드래그로 구간 선택** → 루프 연습

### 재생 컨트롤 (하단 플레이어 바)
| 기능 | 설명 | 단축키 |
|---|---|---|
| 재생/일시정지 | | `Space` |
| 처음으로 | | `Backspace` |
| 속도 조절 | 15~175%, 음정 유지 | `S`(패널), `Shift+1~8`(프리셋), `Shift+A/D`(±1%) |
| 루프 | 드래그 구간 무한 반복 | `L`, `Shift+←/→`(경계 마디 이동) |
| 메트로놈 | | `N` |
| 카운트인 | 재생 전 한 마디 카운트 | `C` |
| 피치 시프트 | ±12반음, 악보는 그대로 오디오만 (신스 전용) | `R` |
| 오디오 소스 | 원본(업로드 오디오) ↔ 신디시스 | `V` |

### 믹서 사이드바 (상시 표시, `T`로 접기 — 믹서/재생/설정 3탭)
- **곡을 열면 전 악기 풀스코어가 기본** — 사이드바에서 악기별로 좁혀보기
- **악보 확대/축소** (우하단 +/− 컨트롤, 50~200%)
- **보기 방식** — 세로 스크롤(기본) ↔ 가로 넘김(◀▶ 페이지 버튼)
- 재생 탭: 속도/루프/카운트인/메트로놈/지판/피치/싱크/마스터 볼륨 (하단 바는 재생·시간·편집만)
- 전 세션(기타/키보드/베이스/드럼 등)을 **한눈에**, 악기 아이콘 표시
- 표기 모드(탭+오선/탭만/오선만)와 내보내기(인쇄/MIDI/.gp)도 사이드바에서 바로
- **S (솔로)** — 그 악기만 듣기 (`Alt+M` = 현재 트랙 솔로)
- **M (뮤트)** — 그 악기 끄기 (`M` = 현재 트랙 뮤트)
- **트랙별 볼륨 슬라이더** (0~150%) — *Songsterr에 없는 기능*
- **👁 악보 표시** — 여러 개 켜면 **멀티트랙 풀스코어 동시 보기**
- **🖨 악기별 인쇄** — 그 악기 악보만 따로 인쇄
- **가사 표시/숨김 토글** (AI 채보 곡에 가사가 있을 때)
- 트랙 이름 클릭 → 그 악기 악보만 보기, **모두 보기** 버튼으로 풀스코어 복귀 (`T` = 사이드바 접기/펴기)
- 마스터 볼륨은 더보기 패널에

### 원본 오디오 (백킹 트랙)
- 오디오 파일(mp3/ogg/wav) 업로드 → **악보와 동기 재생**
- **싱크 조정 패널** — 오프셋(±10초, 인트로 여백 보정) + 템포 배율(80~120%)
- 원본 모드에서는 Songsterr와 동일하게 솔로/뮤트/메트로놈/카운트인/피치 비활성 (통짜 믹스)

### 지판 (Fretboard)
- 플레이어 바 "지판" 버튼 (`F`) — 15프렛 지판에 **재생 중인 노트 실시간 표시** (프렛 번호 포함)
- 현재 트랙 튜닝 자동 반영 (기타 6현 / 베이스 4현), **왼손잡이 모드**
- 키보드/드럼 트랙은 미지원 안내

### 기타 플레이어 기능
- **인쇄** (곡 헤더 🖨) — alphaTab 인쇄 뷰, 다크 모드여도 항상 인쇄용(검정) 색상
- **다운로드** — MIDI, **Guitar Pro(.gp)** (더보기 패널)
- **더보기 설정** — 한 줄에 4마디, 강약 기호 표시, 다크 모드, 표기 모드, 마스터 볼륨
- **다크 모드** — UI + 악보 렌더링 색상까지 반전, localStorage 유지
- **도움말 모달** — 전체 단축키 목록

## 3. 탭 에디터 (`E`)

- **노트 클릭 선택** (초록 박스) → 선택 정보 표시 (마디·현·프렛)
- **프렛 입력** — `0~9`, 두 자리 조합(1→2 = 12프렛, 최대 24)
- **노트 추가** — 빈 위치 선택 후 숫자 입력 / **삭제** — `Del`
- **이동** — `↑↓` 현, `←→` 비트
- **음길이 변경** — 2/4/8/16분 버튼
- **주법 토글** — 팜뮤트(P.M.), 렛링, 비브라토(〜), 데드(×), 고스트(( ))
- 편집 내용은 **재생 오디오에도 즉시 반영**
- 현악기 트랙만 편집 가능 (키보드/드럼 제외)

## 4. 리비전 시스템 (위키 모델)

- **리비전 저장** — 편집 후 GP7 직렬화해 서버 DB에 게시 (로그인 필수, 작성자 기록)
- **리비전 히스토리** — 곡 헤더의 수정 날짜 클릭 → 드롭다운: 버전별 날짜·작성자, 클릭해 과거 버전 열람
- **출처 라벨** — 수동 편집 / 파일 업로드 / AI 생성 배지 (*AI 탭 옥석 가리기 — Songsterr 불만 대응*)
- **정확도 투표** — 리비전별 👍/👎 (로그인 필수, 계정당 1표, 재클릭 취소) (*무검수 문제 보완*)
- **원본으로 되돌리기** — 리비전 전체 삭제 후 시드 복원 (로그인 필수)

## 5. 커뮤니티

- **곡별 댓글** — 채보 오류 제보, 연주 팁 (작성 로그인 필수, 본인 삭제 가능) (*Songsterr에 없는 기능*)

## 6. 계정

- **가입/로그인** — 이메일+비밀번호 (scrypt 해시, httpOnly 세션 쿠키 30일)
- **즐겨찾기(⭐)** — 곡 헤더 별 버튼 → "내 타브" 페이지. 비로그인은 localStorage, **로그인 시 서버 동기화**(합집합 병합, 기기 간 유지)
- 리비전 게시·투표·댓글·빈 탭 생성에 로그인 필요 (열람·재생은 전부 무료/비로그인)

## 7. 콘텐츠 만들기

- **🤖 AI 자동 채보 v2 (2026 SOTA 오픈소스 스택)** — 음악 파일(mp3/wav/ogg)을 넣으면 **악보가 자동 생성**됩니다.
  - **분리**: BS-Roformer-SW 6스템(보컬/기타/피아노/베이스/신스/드럼), **실제 소리가 감지된 악기만** 트랙 생성
  - **채보**: YourMT3+(범용), Transkun(피아노 전용), ADTOF(드럼 킥/스네어/햇/탐/심벌), 특이 음색은 basic-pitch 폴백
  - **음색 판별**: PANNs가 스템을 듣고 기타(어쿠스틱/일렉/디스토션), other 계열(신스/스트링/오르간/브라스 등)을 판별해 **맞는 GM 음색·트랙명** 지정. 신스는 화음/단선율을 **리드/패드 두 트랙으로 분리**
  - **표기**: 조성 감지(Krumhansl) → **조표 자동** + 조성에 맞는 ♭/♯ 표기, **피아노는 C4 기준 큰보표 2단**, 연속 쉼표 병합, 16분음표 비트 그리드 양자화 + 운지 최적화
  - **가사**: faster-whisper(VAD)로 추출 → 곡 페이지 표시 + **보컬 악보의 해당 비트 아래 정렬 배치**
  - **민감도**: 정밀 / 표준 / **최고 정확도**(밀집 구간 다중 시도 앙상블, 2~4배 느림)
  - **BPM 자동 감지**(librosa 비트 트래킹, 직접 입력 가능), 곡 등록 시 AI 채보 배지. (로그인 필수, 3분 곡 기준 약 5분, ML 환경은 `server/ml/.venv` — docs/ml-stack-research.md 참고)
- **빈 탭 만들기** — 헤더 "새 타브" → 제목/아티스트 입력 → 서버에 곡 생성(한글 slug 지원) → 바로 에디터
- **파일 업로드** — `.gp` `.gp3~5` `.gpx` `.musicxml` 등을 플레이어로 열기 (세션 메모리, 모든 플레이어 기능 동작)
- 시드 3곡은 퍼블릭 도메인 자체 편곡 (Ode to Joy · Twinkle Twinkle · Jingle Bells, 각 5트랙 풀밴드)

## 9. 기술·배포

- **웹**: Vite + React + TS + alphaTab (렌더+신스+커서 싱크)
- **API 서버**: Node 24 내장 모듈만 (`node:http` + `node:sqlite`), 의존성 0개, 프로덕션에서 정적 서빙 겸함
- **배포**: Docker 단일 컨테이너(Railway/Fly/Render용 Dockerfile 포함) / Vercel 읽기 전용 데모(서버리스 함수 1개)
- API 명세·실행 방법은 [README](../README.md), Songsterr 대비 구현 현황은 [feature-parity.md](feature-parity.md) 참고
