# Songsterr 콘텐츠 제작 시스템 (에디터·리비전·AI 채보·저작권)

> 조사일: 2026-07-08 | 조사 방법: songsterr.com 공식 페이지(help/about/plus/new), 포럼(TalkBass, BassBuzz), 언론(heise), 웹 검색
> 표기: **[확인됨]** = 공식 페이지·복수 소스에서 직접 확인 / **[추정]** = 간접 증거 기반 또는 미공개 정보

---

## 1. 탭 에디터 (Tab Editor)

- **[확인됨]** 웹 플레이어에 인라인 에디터가 내장되어 있음. `E` 키 또는 우측 하단의 에디터 아이콘(연필)으로 활성화. 노트를 클릭하면 컨텍스트 편집 메뉴가 열리고, 방향키로 노트 간 이동, 숫자 `0~9` 입력으로 프렛 변경, `Backspace`로 삭제. 로그인 상태에서 수정 사항은 **자동 저장**됨.
- **[확인됨]** Guitar Pro 기반 워크플로우를 공식 지원: 기존 리비전을 **다운로드 → Guitar Pro/TuxGuitar에서 오프라인 편집 → 재업로드**하는 흐름을 help에서 안내. 즉 내부 포맷은 Guitar Pro 파일과 상호 호환.
- **[확인됨]** 신규 탭 생성 시 **.gp(Guitar Pro) 파일 업로드**가 공식 경로 중 하나.
- **[확인됨]** 렌더링/재생 엔진이 지원하는 표기: 해머온, 풀오프, 벤드/릴리즈 벤드, 슬라이드/레가토 슬라이드, 비브라토/와이드 비브라토, 렛링, 팜뮤트, 고스트/데드 노트, 업/다운 스트로크, 내추럴/핀치 하모닉스, 폴리포니(다성부), 템포 변경, 박자표 변경 등 (howtoreadtab 페이지 + "bend editor", "Tempo Change", "Time signature changes" 테스트 탭 존재).
- **[추정]** 위 표기 전부를 웹 에디터 UI에서 직접 입력·수정할 수 있는지는 미확인. 에디터 튜토리얼 탭(`/a/wsa/editor-tutorial-tab-s576897`)이 존재.
- **[확인됨]** 편집은 **공개(public) 수정** 또는 **비공개(private) 수정** 중 선택 가능 — 남의 탭을 고쳐 공개 리비전으로 올리거나, 자기만 보는 사본으로 유지.
- **[확인됨]** 탭 삭제: Edit 모드에서 제목 옆 휴지통 아이콘 → 확인. **생성자만** 삭제 가능, 영구 삭제.

## 2. 리비전 시스템

- **[확인됨]** 곡 페이지 제목 아래 **"Revision from: [날짜]"** 링크(= 곡 페이지의 "수정 날짜")를 클릭하면 해당 곡의 **리비전 목록**이 열림. 임의 리비전을 다운로드해 개선 후 새 리비전으로 업로드 가능. 이전 버전으로 **되돌리기(revert)** 도 이 화면에서 수행 (웹 앱 전용).
- **[확인됨]** 편집 권한 모델은 **위키(Wikipedia) 방식**: "누구나 새 탭을 올리고 기존 탭을 수정할 수 있다." 사전 승인 절차 없이 즉시 반영되며, "소프트웨어가 실수를 쉽게 되돌릴 수 있게 하고, 숙련된 에디터들이 지켜보면서 편집이 누적적 개선이 되도록 보장"하는 **사후 교정 모델**.
- **[확인됨]** 리비전 삭제: **자신이 만든 마지막 리비전만** 삭제 가능, 영구 삭제.
- **[추정]** 별도의 승인 대기열이나 staff 검수는 없는 것으로 보임. 과거 리뷰(2010년대)에는 오류 리포트 추천/비추천 투표, 모더레이터의 부적절 리포트 삭제 도구가 있었다고 기록되어 있으나 현재 UI 유지 여부는 미확인.

## 3. AI 전사 (Songsterr AI)

- **[확인됨]** 입력: **YouTube 링크 붙여넣기 또는 오디오 파일 업로드** (`songsterr.com/new`). 문구: "Paste a YouTube link and let AI create accurate guitar, bass, and drum tabs in minutes."
- **[확인됨]** 생성 전 설정 옵션: 박자표(기본 Auto), 픽업(못갖춘)마디, 템포/BPM, 셋잇단(triplet feel), 트랙(악기) 선택, 카포 위치.
- **[확인됨]** 지원 악기/트랙: **보컬(멜로디), 리듬 기타, 리드 기타, 베이스(4현), 드럼**.
- **[확인됨]** 쿼터·가격: **Plus에 월 50곡 전곡(full tab) 전사** 포함. 처리 시간은 수 분~하루. 무료 사용자는 **20초 길이 제한** (heise 보도).
- **[확인됨]** AI 결과물은 사용자 대시보드(My Tabs)에 초안으로 생성되며, **공개(publish) 전에 정확도를 직접 검수하라고 요구**함. AI 탭도 일반 리비전과 같은 커뮤니티 공개 흐름을 탐.
- **[확인됨]** 출시 시점: 공식 발표 없이 조용히 롤아웃. 포럼(2025년 2월)·heise 기사(2025년 4월) 기준 **2024년 말~2025년 초 출시**로 추정.
- **[확인됨]** 품질 평판: "hit and miss"지만 시작점으로는 충분. 클래식 록/메탈 리프는 상당히 정확. 약점: 베이스 트랙 정확도 최저, 팜뮤트·빠른 슬라이드 누락, 멀티 기타 트랙 분리 부정확, 불필요한 포지션 이동. 범용 LLM 대비 월등하다는 평가(heise).
- **[추정]** 기반 기술 비공개. 처리 시간과 트랙별 분리 전사로 볼 때 소스 분리(demucs류) + 악기별 AMT(automatic music transcription) 파이프라인의 서버 배치 구조로 추정.

## 4. 코드·가사 기능

- **[확인됨]** 공식 help: "**코드 기호와 가사는 transcriber(채보자)가 추가**한다. 추가하는 사용자가 많지 않다. 있으면 플레이어에 표시된다." → 별도 코드/가사 DB가 아니라 **탭 파일(Guitar Pro 포맷) 안에 포함된 코드·가사 데이터를 렌더링**하는 구조.
- **[추정]** 입력 경로는 주로 Guitar Pro 데스크톱에서 코드/가사를 넣어 업로드하는 방식. 웹 에디터에서 직접 입력 가능 여부 미확인.
- **[추정]** 전용 "코드/가사 뷰 모드"가 있다는 공식 문서는 없음. 악보 위 코드 기호 + 음표 아래 가사가 함께 렌더링되는 형태.

## 5. 저작권·라이선스

- **[확인됨]** 공식 입장(help): "**Songsterr는 100% 합법이다. 원곡 창작자(퍼블리셔)와 라이선스를 맺고 있으며, 수익의 거의 절반을 로열티로 지급한다.**" 운영사는 미국 법인 Guitar Tabs LLC.
- **[확인됨]** 일부 탭이 없거나 내려간 이유: "해당 아티스트의 **퍼블리셔와 아직 라이선스가 없기 때문**" → 라이선스 미확보 곡은 숨김/차단 처리.
- **[확인됨]** ToS에 DMCA 지정 대리인 절차와 반복 침해자 계정 해지 조항 존재.
- **[확인됨 - 업계 배경]** 2005~2006년 MPA/NMPA가 탭 사이트에 대대적 법적 조치 선언. Ultimate Guitar는 2010년 Harry Fox Agency와 라이선스 계약 체결. 이후 "퍼블리셔 계약 + 로열티 지급" 모델이 업계 표준.
- **[추정]** Songsterr의 구체적 계약 상대·조건은 비공개. "사용자가 직접 채보한 것만 받고 퍼블리셔 공식 악보 사본은 배포하지 않는다"는 원칙 + 퍼블리셔별 개별 라이선스 조합으로 운영 추정. 대규모 소송·강제 폐쇄 기록 없음(개별 곡 삭제만 존재).
- **⚠️ 클론 프로젝트 시사점**: 탭(채보물)도 원곡의 2차적 저작물이라 퍼블리셔 라이선스 없이 공개 서비스하면 침해 소지가 큼. "사용자 생성"만으로 면책되지 않음(DMCA 세이프하버는 신고 대응 체계 전제). Songsterr/UG 모두 유료화 수익 → 로열티 지급 구조로 합법화함. **개인 학습·포트폴리오용이라면 자작/퍼블릭 도메인 곡 또는 비공개 운영을 권장.**

## 6. "New tab"(새 탭) 생성 플로우

- **[확인됨]** 상단 툴바 **"New Tab" 버튼** → 두 갈래:
  1. **빈 탭 생성** → 웹 에디터에서 처음부터 작성. YouTube 링크를 걸면 "Songsterr AI가 몇 분 내에 탭 생성 시도".
  2. **Guitar Pro 파일 업로드** → 곡 메타데이터(아티스트/곡명 등) 입력 후 제출.
- **[확인됨]** `/new` 페이지(AI 생성기)는 비로그인으로도 옵션 설정까지 가능하지만, **AI 실행·탭 생성·업로드·편집 저장은 로그인 필수** (무료 계정 가능).
- **[확인됨]** 탭은 **비공개(Private) 상태로 생성** 가능 — 게시하지 않은 개인 탭은 생성자만 접근. 관리는 `/mytabs`.

## 7. 원음(Original Audio) / 백킹 트랙 / 플레이스루

- **[확인됨]** Plus 기능 **"Pause-Free Sync with Original Audio"**: 탭을 **원음, 백킹 트랙, 플레이스루** 중 하나와 동기화 재생. 무료 사용자는 원음 재생 시 **10마디마다 일시정지**(신스 재생은 무제한), Plus는 무중단.
- **[확인됨]** 원음이 없는 곡은 내장 신디사이저(MIDI 기반) 재생으로 폴백. Pitch Shift는 "신스 오디오"에만 적용 명시 → 원음/신스 이원 재생 구조.
- **[추정]** 원음 확보 경로는 비공개. 곡별로 원음 유무가 갈리는 점으로 보아 **퍼블리셔·레이블별 개별 계약분만 원음 제공**으로 추정. 사용자 업로드 원음 증거 없음.
- **[추정]** 탭-원음 싱크는 내부 툴로 마디 단위 타임맵을 만드는 방식으로 보임 ("Tempo changes sync fix" 내부 테스터 탭 존재 → 템포 변화 구간 싱크를 수동/반자동 보정하는 도구가 있는 듯).
- **[추정]** "backing track"/"playthrough"는 원곡 마스터를 못 쓰는 곡의 대체 콘텐츠(커버 연주 음원)로 보임.

## 클론 설계 관점 핵심 요약

1. **데이터 모델 = Guitar Pro 호환 포맷 + 곡 단위 리비전 스택** (리비전은 append-only, 마지막 자기 리비전만 삭제, 임의 시점 revert).
2. **모더레이션 = 사전 승인 없음, 사후 revert + 숙련 에디터 감시** (위키 모델). 라이선스 미보유 곡 노출 차단이라는 별도 축 존재.
3. **AI 전사는 별도 배치 파이프라인** (YouTube/오디오 → 트랙별 전사 → 사용자 초안 → 수동 검수 후 공개), Plus 수익화 장치(월 50곡).
4. **법적 기반이 서비스의 전제**: 퍼블리셔 라이선스 + 수익 배분 없이는 콘텐츠 축·원음 축 모두 성립 불가.
