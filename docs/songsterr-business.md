## 1. 계정 시스템

- **가입 방법**: Google OAuth("Continue with Google"), Apple OAuth("Continue with Apple"), 이메일+비밀번호 3가지. 가입 폼 필수 필드는 이름/이메일/비밀번호 + 약관 동의. Facebook 로그인 없음. — **확인됨** (signin/signup 페이지 직접 확인. 단, /signup 페이지 자체엔 이메일 폼만 보이고 소셜 버튼은 /signin에서 확인됨)
- **무료 계정 역할**: 가입 없이도 무료 플랜 전체(탭 열람+기본 MIDI 재생, 검색) 사용 가능. 계정은 (1) 즐겨찾기(Favorites) 기기 간 동기화, (2) 탭 제작/기여(New tab, 탭 퍼블리시)에만 필요. — **확인됨** (공식 help)
- **프로필 기능**: 별도의 공개 프로필/팔로우 같은 소셜 기능은 발견되지 않음. 계정 페이지는 구독 관리(카드 변경, 1클릭 해지) 중심. 계정 삭제는 지원팀 문의 방식. — **확인됨** (help 기준; "소셜 기능 없음"은 **추정**)

## 2. 구독 (Songsterr Plus)

- **가격**: 웹 기준 **$9.90/월** 단일 플랜. 공식 FAQ는 "월간 결제만 있고 평생(lifetime)/연간 플랜 없음"이라고 안내. — **확인됨**
  - 일부 쿠폰 사이트에 "연간 $99.99 (37% 절약)" 언급이 있으나 공식 페이지에서 확인 불가 — **추정(신뢰도 낮음, 공식 FAQ와 상충)**
- **iOS 인앱 가격**: **$9.99/월** (App Store 가격 티어 때문에 웹과 10센트 차이), 레거시 구독 $4.99/월 항목 잔존. — **확인됨** (App Store 인앱 구매 목록)
- **무료 체험**: 현재 없음. 과거(2012년경) 14일 무료 체험 프로모션 이력만 존재. — **추정** (공식 plus/help 페이지에 체험 언급 전무 + 서드파티 확인)
- **지역별 가격/학생 할인**: PPP 기반 지역 차등가나 상시 학생 할인 프로그램은 발견되지 않음. — **추정** (부재 증거 기반)
- **결제 채널**: 웹 = Songsterr 자체 결제(PayPal, Apple Pay, Visa/MC/Amex/Discover/JCB), iOS = Apple 인앱, Android = Google Play. **한 구독으로 웹+iOS+Android 전부 이용 가능**(단일 로그인). 과거 Android에는 일회성 "Premium" 구매가 있었고, 이 구매자는 크로스 플랫폼 1개월 무료 보상 정책 있었음. — **확인됨**
- **환불**: 웹 결제는 "불만족 시 100% 환불" 명시. iOS 결제는 Apple 환불 절차로만 가능(Songsterr가 직접 처리 불가). 해지는 계정 페이지에서 1클릭. — **확인됨**

## 4. 검색 & 디스커버리

- **검색**: 곡명/아티스트 자동완성 검색(부분 일치 지원). **가사 검색은 지원 근거 없음** — **확인됨 / 가사 미지원은 추정**
- **필터**: 악기(Any/Guitar/Bass/Drums), 튜닝, **난이도** 드롭다운 필터. 로그인 불필요. — **확인됨**
- **난이도**: 탭에 난이도 등급이 붙고 필터링 가능. 산정 방식은 알고리즘 자동 계산으로 보임 — **필터 존재 확인됨 / 산정 방식은 추정**
- **인기/트렌딩**: 홈에 인기 탭 카드 리스트 + `/tags/<장르·국가>` 형태의 큐레이션 태그 페이지(예: /tags/spanish, /tags/france). — **확인됨**
- **Chords vs Tab**: Ultimate Guitar식 "코드+가사 시트"라는 별도 콘텐츠 타입은 없음. 탭은 전부 악보형 인터랙티브 탭이며, **채보자가 넣은 경우에만 코드 기호·가사가 탭 위에 표시**됨. — **확인됨**
- 콘텐츠는 100% 사용자(채보자) 기여 아카이브 — 곡 약 40만, 탭 트랙 약 130만 개. — **확인됨**

## 7. 경쟁사 대비 포지셔닝

Songsterr의 차별점은 **"무료로도 카탈로그 전곡을 인터랙티브 멀티트랙 악보 플레이어로 볼 수 있다"**는 것이다. Ultimate Guitar는 카탈로그가 훨씬 크지만 무료 콘텐츠 대부분이 텍스트 탭/코드 시트이고 인터랙티브 기능은 Pro에 묶여 있으며, Guitar Pro/mySongBook은 데스크톱 편집기 구매 + 곡당/구독 악보 판매 모델이고, Yousician은 마이크 피드백 기반 게임화 학습 앱이라 정확한 원곡 탭 아카이브가 아니다. Songsterr는 학습 커리큘럼·게임화·텍스트 탭 없이 "정확한 악보형 탭 + 리얼 사운드 동기 재생" 단일 경험에 집중하고, 유료는 속도조절·루프·솔로/뮤트·원음 싱크·인쇄 같은 연습 편의 기능만 잠근다. 리뷰어들의 공통 평가는 "무료 경험은 Songsterr 승, 유료 가성비는 UG 승". — **확인됨** (Guitar Chalk, Singular Sound 비교 리뷰)
