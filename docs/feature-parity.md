## ❌ 미구현 (이유별)

**API/인프라 한계 — 외부 서비스 필요**
- **음성 제어(Voice Control)**: Songsterr는 OpenAI Realtime API 사용. API 키·비용 필요 → 원하면 붙일 수 있는 구조(플레이어 함수가 전부 노출되어 있음)
- **원곡 YouTube 싱크**: 라이선스·YouTube API 이슈. 파일 업로드로 대체함
- **백킹/솔로 스템 분리**: 음원 분리 모델(demucs/MVSEP) 서버 필요
- **AI 채보**: 오디오→탭 자동 생성. 대형 ML 파이프라인

**alphaTab 1.8 API 부재**
- **전조(Transpose, 악보 표기 이동 ±36)**: 오디오 전조(`changeTrackTranspositionPitch`)만 지원 — 피치 시프트로 구현됨. 표기 전조는 노트 재배치 로직을 직접 짜야 함
- **메트로놈 보이스 카운팅("one-two")**: alphaTab 메트로놈은 클릭음만

**작은 갭 (추가 가능, 우선순위 낮음)**
- 루프 경계 Shift+←/→ 이동, Alt+드래그 루프 이동 (드래그 재선택으로 대체 가능)
- 루프 사이 카운트인 옵션
- 멀티레스트 표시 설정, 지판 음이름/12프렛 옵션
- Sheet Music/Lead Sheet 표기 모드, Chords 페이지(가사+코드 뷰, 카포, 자동 스크롤)
- 코드 감지(탭 화성 분석 → 코드 자동 표기)
- 트랙 자동 전환(클린↔디스토션)
- 빈 탭 새로 만들기(현재는 파일 업로드만), 에디터 마디 추가/삭제
- 플레이리스트, 아티스트 페이지, 태그 큐레이션, 난이도 표시
- Google/Apple OAuth

**의도적으로 제외**
- Plus 과금/광고/10마디 제한 (클론 목적상 전 기능 무료)
- 오프라인 캐시 (Songsterr도 모바일 앱 전용)
- MP3/WAV 다운로드 (원곡 오디오가 없으므로)
