# Songsterr 서비스 분석 문서 (클론 개발용)

> 검증일: 2026-07-08 · 출처: songsterr.com 본문, /plus, /help(FAQ) 및 외부 리뷰
> 목적: songsterr.com 클론 웹 개발을 위한 기능 명세의 기초 자료

---

## 1. 서비스 개요

Songsterr는 기타·베이스·드럼 악보(Tab)를 **재생 가능한 대화형 플레이어**로 제공하는 학습 플랫폼이다.
정적인 텍스트 탭이 아니라, 리듬 정보가 포함된 악보를 오디오와 동기화해 재생하는 것이 핵심 차별점이다.

- **라이브러리 규모**: 약 40만+ 곡, 130만+ 탭, 6만+ 아티스트 (마케팅 문구의 "100만+ 탭"은 탭 기준 수치)
- **플랫폼**: 웹, iOS, Android
- **수익 모델**: 프리미엄(freemium) — 무료 + Plus 구독

## 2. 핵심 기능 (검증 완료)

### 2.1 대화형 플레이어
- 재생 중 현재 연주 위치의 음표가 실시간 하이라이트(커서 이동)
- 곡 하나에 여러 트랙(악기별) 존재 — Instrument 버튼으로 트랙 전환
- 신스(Synth) 재생 + **원곡 오디오 동기화 재생** 지원
  - ⚠️ 무료 사용자는 원곡 오디오 동기화 시 **10마디마다 일시정지** 발생. Plus는 pause-free
- 백킹 트랙(backing track), 플레이스루(playthrough) 오디오도 제공

### 2.2 오디오 믹서 (Solo / Mute)
- 트랙별 Solo(해당 악기만 듣기) / Mute(끄기)
- ⚠️ **Plus 전용 기능** (무료 아님)

### 2.3 속도 조절 (Playback Speed)
- 음정(pitch) 변화 없이 템포만 조절
- **Plus 전용**

### 2.4 피치 시프트 (Pitch Shift)
- 악보(탭 표기)는 그대로 두고 오디오 음정만 이동 — 변칙 튜닝 기타에 맞출 때 사용
- **Plus 전용**

### 2.5 루프 (Loop)
- 마디 구간을 지정해 무한 반복 재습
- **Plus 전용**

### 2.6 메트로놈 / 카운트인
- 메트로놈 켜기/끄기 (단축키 N), 카운트인 (단축키 C)

### 2.7 오프라인 캐시
- ⚠️ **모바일 앱(iOS/Android) 전용**. 웹은 온라인 전용(online-only)
- 앱에서는 한 번 연 곡이 캐시되어 오프라인에서도 작동 (곡을 끝까지 재생할 필요 없음)

### 2.8 가사 및 코드
- ⚠️ 모든 곡 제공이 아님 — **채보자(transcriber)가 코드/가사를 입력한 곡만** 표시됨
- 공식 FAQ: "코드 기호와 가사가 채보에 포함된 경우에만 표시되며, 추가하는 사용자가 많지 않다"

### 2.9 탭 에디터 / 사용자 채보
- 사용자가 직접 탭을 만들고 수정(revision) 가능 (단축키 E로 에디터 진입)
- 탭마다 리비전 이력 존재

### 2.10 AI 채보 (AI Transcription) — 비교적 신규 기능
- 오디오에서 탭 자동 생성. **Plus: 월 50개 완곡 채보**
- (사용자 요약에는 없던 기능 — 클론 범위에 넣을지 결정 필요)

### 2.11 다운로드 / 인쇄
- **Plus**: MIDI, Guitar Pro, MP3, WAV 다운로드 / 고품질·고대비 탭 인쇄 (전체 또는 선택 페이지)

### 2.12 즐겨찾기 (My Tabs / Favorites)
- 무료 계정 생성 시 기기 간 즐겨찾기 동기화. 비로그인 시 복구 불가

## 4. 키보드 단축키 (웹 플레이어)

| 키 | 동작 |
|---|---|
| `Space` | 재생 / 일시정지 |
| `Backspace` | 곡 처음으로 이동 |
| `←` / `→` | 커서 뒤로 / 앞으로 |
| `↑` / `↓` | 줄(line) 위 / 아래 이동 |
| `T` | 트랙 목록 표시 |
| `S` | 재생 속도 변경 |
| `L` | 루프 모드 켜기/끄기 |
| `Shift + →` / `Shift + ←` | 루프 오른쪽 경계 이동 |
| `N` | 메트로놈 켜기/끄기 |
| `C` | 카운트인 켜기/끄기 |
| `M` | 현재 트랙 음소거/해제 |
| `Alt + M` | 솔로 모드 켜기/끄기 |
| `E` | 탭 에디터 활성화 |
| `V` | 오디오 소스 전환(원본/신스) |
| `R` | 피치 시프트 |
| `K` | 전조(Transpose) |
| `Shift + 1~8` | 속도 프리셋(15~175%) |
| `Shift + A` / `Shift + D` | BPM ±1 미세 조정 |

> 전체 검증판 단축키·플레이어 세부 동작은 [songsterr-player.md](songsterr-player.md) 참고.

## 5. 사이트 구조 (IA)

- `/` — 메인: 인기 곡 목록(60여 곡 노출), 검색 진입점
- `/plus` — Plus 요금제 소개·업그레이드
- `/help` — FAQ
- `/mytabs` — 즐겨찾기 (My Tabs)
- 곡 페이지 — `/a/wsa/{artist}-{song}-tab-s{id}` 형태의 URL, 곡별 플레이어
- 검색, 악기 필터(Any / Guitar / Bass / Drums), 탭 만들기(New tab), 로그인

## 6. 곡 페이지 UI 상세 (스크린샷 기준, 한국어 UI)

> 기준 화면: Enter Sandman (Metallica) 곡 페이지, 비로그인 무료 상태, 데스크톱 웹

### 6.1 상단 헤더 (고정 내비게이션)

좌측 로고 + 우측으로 아이콘·라벨 세트가 가로로 배치된다.

| 항목 | 아이콘 | 비고 |
|---|---|---|
| Songsterr 로고 | 파란 사각형 로고 | 홈으로 이동 |
| SONGSTERR PLUS | `+` (초록) | 업그레이드 페이지 진입, 초록색으로 강조 |
| 검색 | 돋보기 | |
| 내 타브 | 별 | 즐겨찾기 목록 |
| 새 타브 | 문서+ | 탭 만들기(에디터) |
| 도움말 | 물음표 | FAQ |
| 로그인 | 화살표 | |

### 6.2 곡 정보 영역 (타이틀)

- 곡 제목: **"Enter Sandman 탭"** (대형 세리프체)
- 그 아래 한 줄: ⭐ 즐겨찾기 버튼 · "탭" 드롭다운(악보 종류 선택) · **수정 날짜: 2026. 7. 6.**(리비전 날짜) · 🖨 인쇄 버튼(**초록 자물쇠 배지 = Plus 잠금**)
- 아티스트명 **Metallica** (링크, 세리프체)

### 6.3 악보(탭) 렌더링 영역

- 섹션 라벨: "인트로", "인트로 2" 등 구간 이름 표시
- 템포 표시: ♩ = 123
- 좌측에 기타 튜닝(E B G D A E) + 박자표(4/4)
- 마디 번호가 각 마디 위에 표시 (1, 2, 3 …)
- **초록색 재생 커서**(세로 바)가 현재 위치 표시 — 재생 시 이동하며 하이라이트
- 여러 줄(row)로 탭이 이어짐, 벤드·슬라이드·서스테인 곡선 등 주법 기호 렌더링
- 마디 근처에 `⋮` 컨텍스트 메뉴 버튼 (구간 관련 동작)
- 무료 사용자에게는 악보 중간에 **배너 광고** 삽입 (닫기 X 버튼 있음)

### 6.4 하단 플레이어 바 (고정)

좌→우 순서:

| 컨트롤 | 표시 | 잠금 여부 |
|---|---|---|
| 트랙 선택기 | 악기 아이콘 + "디스토션 기타 / Kirk Hammett \| Gibson Les…" + `^` 펼침 | 무료 |
| 재생 버튼 | 초록 ▶ | 무료 |
| 오디오 소스 토글 | **원본 / 신디시스** 2단 스위치 | 원본 재생 자체는 무료(10마디 제한) |
| 속도 | 달리기 아이콘 + "100%" | Plus |
| 루프 | 순환 화살표 | Plus |
| 솔로 | 헤드폰 | Plus |
| 뮤트 | 스피커X | Plus |
| 카운트인 | ③ | 무료 |
| 메트로놈 | 메트로놈 아이콘 (비활성 상태로 보임 — 원본 오디오 모드에선 비활성 추정) | 무료 |
| 음성 | 마이크 아이콘 + **초록 자물쇠** | Plus (**Voice Control** — 핸즈프리 음성 명령으로 플레이어 조작. OpenAI Realtime API 기반. 상세: songsterr-player.md §1) |
| 다운로드 | 다운로드 아이콘 + **초록 자물쇠** | Plus |
| 더보기 | ⋮ | — |
| 편집기 | 연필 | 무료(에디터 진입) |

- **Plus 잠금 표시 패턴**: 해당 아이콘 우하단에 초록색 자물쇠 배지
- **더보기(⋮) 메뉴** 내용: "코드 감지", "전조(Transpose)" ›, "설정" › — 서브메뉴로 이어짐

### 6.5 클론 구현 관점 메모

- 페이지 레이아웃은 3층 구조: **고정 헤더 / 스크롤되는 악보 본문 / 고정 플레이어 바**
- 악보는 SVG/Canvas 렌더링 대상 (마디 단위 레이아웃 + 커서 애니메이션)
- Plus 잠금은 기능 숨김이 아니라 **자물쇠 배지 + 클릭 시 업셀** 패턴
- UI가 브라우저 언어에 따라 현지화됨(한국어 노출 확인) — i18n 고려
